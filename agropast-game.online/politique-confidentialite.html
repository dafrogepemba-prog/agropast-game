Politique de Confidentialité

1. Données collectées

AgroPast Game collecte les données suivantes lors de l'inscription :

  • Numéro de téléphone WhatsApp (identifiant de connexion)
  • Pseudo / nom d'utilisateur
  • Adresse e-mail (optionnelle)
  • Pays de résidence
  • Scores et statistiques de jeu

2. Finalité du traitement

Les données collectées sont utilisées exclusivement pour :

  • Gérer votre compte et votre progression dans le jeu
  • Alimenter le classement mondial des joueurs
  • Traiter les demandes de retrait de gains
  • Améliorer l'expérience de jeu

3. Publicités Google AdSense / AdMob

AgroPast Game utilise Google AdSense pour la version web et Google AdMob pour la version Android. Ces services peuvent collecter des données anonymisées pour afficher des publicités pertinentes. Vous pouvez consulter la politique de confidentialité de Google à l'adresse : policies.google.com/privacy.

Les publicités récompensées (Rewarded Ads) ne sont diffusées qu'avec votre consentement explicite, lorsque vous appuyez sur le bouton correspondant dans le jeu.

4. Cookies et stockage local

Sur la version web, AgroPast Game utilise le stockage local du navigateur (localStorage) pour sauvegarder votre session et votre progression de jeu. Ces données ne sont pas transmises à des tiers.

5. Durée de conservation

Vos données de compte sont conservées tant que votre compte est actif. Vous pouvez demander la suppression de votre compte et de vos données à tout moment via la page contact.

6. Vos droits

Conformément au RGPD et aux lois en vigueur, vous disposez d'un droit d'accès, de rectification, de suppression et d'opposition concernant vos données personnelles. Contactez-nous via la page contact pour exercer ces droits.

7. Contact

Pour toute question relative à la confidentialité de vos données, consultez notre page contact.
